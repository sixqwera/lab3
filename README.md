# Задание 2

## Задача 1

Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для
последовательности(Seq.fold)

### Алгоритм решения

Запуск: Программа выводит запрос «Вводите цифры:».
Ввод данных (Рекурсия): Пользователь вводит символы по одному.
Если введена одна цифра (0–9), она сохраняется в список, и программа ждет следующую.
Если введено несколько символов, буква или пустота с пробелом, выводится «Ошибка!», и ввод повторяется.
Остановка: Для завершения ввода пользователь нажимает Enter (пустая строка).
Обработка (Match): Каждая цифра из списка сопоставляется со словом (от «ноль» до «девять»).
Сборка строки (Fold): Программа проходит по списку и склеивает все слова в одну текстовую строку через пробел.
Вывод: В консоль выводится исходный числовой список и итоговая строка с названиями цифр.

# Задание 3

## Задача 1

### Текст задачи

Вывести список файлов, расширение которых начинается с заданного
символа, в указанном каталоге.

### Алгоритм решения

Запрос пути: Программа просит ввести путь к папке.
Проверка директории: Если путь не существует, выводится nah и работа завершается.
Ввод буквы: Если путь верный, программа просит ввести символ для фильтрации расширений.
Валидация символа:
Если введена цифра или пустая строка — выводится ошибка.
Если введена буква — программа берет её первый символ (target).
Поиск файлов (EnumerateFiles): Программа сканирует указанную папку и получает список всех файлов.
Фильтрация (Seq.filter): Для каждого файла извлекается расширение. Проверяется, что оно длиннее одного символа и его первая буква (после точки) совпадает с введенной.
Вывод:
Если подходящих файлов нет — выводится сообщение о пустом результате.
Если файлы найдены — их названия выводятся списком через тире.
